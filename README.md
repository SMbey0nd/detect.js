# Detect.js

## 统一环境检测包。
当前版本：V0.2.1

---

## detect
环境检测核心文件(core)

包含以下组件(plugin)：

## Network
网络状况检测
## UA
用户代理检测
## Ability
能力检测(incoming)
## Hardware
硬件检测(incoming)
## DeviceAPI
设备API检测(incoming)

---

## 用法
### 1.引入detect.js文件
### 2.引入需要的组件
### 3.运行
```js
  	DETECT.init();
```
